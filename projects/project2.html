Cybersecurity Log Analyzer

Status: Coming Soon

Project Summary

This project analyzes system logs to detect threats, failed logins, anomalies, and generate security alerts — similar to what a SOC analyst or cybersecurity auditor does.

Features (Planned)

  • Parse Windows/Linux log files
  • Failed login detection
  • Brute-force attack detection
  • Anomaly score generator
  • HTML report output

Technologies

  • Python (Regex, Pandas)
  • JSON or Syslog format logs
  • Front-end visualization
